# FaceScore
###数据分析
obj格式有4种数据，分别以以下字母开头：v顶点，vt纹理坐标，vn顶点法向量，f面  
有用的其实只要定点v，因为vt纹理坐标和vn顶点法向量都是0，  
f面为v/vt/vn （顶点索引 / 纹理坐标索引 / 顶点法向量索引）也没什么意义，  
**原来没有颜色是因为没有将原图的颜色映射回去**  
所以训练的时候只需要用v即可  
v的长度为43867,发现用一半的数据看起来也比较真实，所以间隔取样了一般的点来训练，间隔取样效果在Docs目录下
